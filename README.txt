INTRODUCTION
============

***WARNING: ShellOL IS INTENTIONALLY VULNERABLE.
DO NOT USE ON A PRODUCTION WEB SERVER. DO NOT
EXPOSE shellOL IN AN UNTRUSTED ENVIRONMENT.***

ShellOL is a configurable shell command injection testbed. shellOL allows
you to exploit shell command injection flaws, but furthermore allows
a large amount of control over the manifestation of the flaws.

ShellOL is based on the idea of SQLol, an earlier release which
allows for SQL injection exploitation.


REQUIREMENTS
============

PHP 5.x
Web server

USAGE
=====

Place the ShellOL source files on your Web server and
open in a Web browser.
